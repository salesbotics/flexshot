Everything you need after “wait, let me screenshot this”

From a quick tab grab to a narrated walkthrough — Flexshot keeps capture and editing in one flow.

Screenshot modes

Visible area, drag-to-select region, or full-page scroll stitch. Shortcuts: Alt+Shift+V / R / F.

Full annotation

Pen, marker, arrows, shapes, text, numbered pins, blur/redact, and crop — with undo and zoom.

Screen + audio

Record a tab, window, or screen. Mux mic and system audio, then download video and a separate audio file.

Web → PDF

Turn the live page into a multi-page PDF download for archives, tickets, and async reviews.

Export anywhere

Download PNG, JPG, or WebP from the editor. Copy annotated images to the clipboard in one click.

Built for teams

Perfect for bug reports, design feedback, support replies, and product demos without leaving Chrome.

Three steps. Zero friction.

  1. Capture Pick visible, region, full page, record, or PDF from the popup.
  2. Annotate Mark what matters — draw, type, pin, blur sensitive bits, crop the rest.
  3. Ship Download the image, video, audio, or PDF and drop it into Slack, email, or a ticket.

Install Flexshot

Load the unpacked extension while we prepare the Chrome Web Store listing.

  1. Open chrome://extensions
  2. Enable Developer mode
  3. Click Load unpacked and select the flexshot folder
  4. Pin Flexshot and capture your first shot

Chrome Web Store ready: package the folder (exclude marketing if you prefer), upload icons (16/48/128), and use the listing copy below.
